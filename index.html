Securent

A modern car rental platform, where drivers get growing discounts on future rides by persistently obeying the speed limits and cars' owners can make confident and reasonable decisions about giving permission to rent their vehicle to a particular driver

  • Insiration and philosophy

Easily rent as a driver

Just pick a car available, wait for a positive response and lock / unlock with your phone! No more passing the key in person!

Be confident as an owner

You choose whether you provide access to your car or not depending on the previous history and overall rating of the user. We obtain and constantly update this infornation by analyzing the way the user obeys the speed limits.

Get growing savings on future rides

If you constantly obey speed limits, our application will be giving you growing discounts in order to motivate you to drive safe and carefully.

How we built it

This is a short description of our technology stack and APIs that we used in our product

Node.js Server

JavaScript is our primary programming language. We used Node.js Express to create a web application and we deployed it on Google Cloud App Engine

Smartcar

We use smartcar API to access cars' location, info, odometer data, and perform locks / unlocks

AWS Amplify

We used React Application rendered with the help of AWS Amplify to create a basic workflow and call various hand-made APIs

Google Cloud App Engine

We used Google Cloud App Engine to deploy our application in the cloud and to perform Google Maps API queries

AWS Aurora

We store our data in AWS Aurora database that talks to Node.js cloud server and React Application

Google Maps API

We used that API to retrieve the speed limit based on latitude and longitude of the car at a certain point of time
